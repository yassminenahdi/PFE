# Exemples de diagrammes pour le rapport PFE

---

## 1. Diagramme d’architecture globale

[Utilisateur]
     |
     v
[Frontend Angular] <----> [Backend Spring Boot] <----> [Microservice ML (Python/Flask)]
                                         |
                                         v
                                 [Base de données]

---

## 2. Diagramme de séquence (flux d’une transaction)

Utilisateur -> Frontend : Saisie de la transaction
Frontend -> Backend : Envoi des données de transaction
Backend -> Microservice ML : Requête de prédiction de fraude
Microservice ML -> Backend : Réponse (fraude ou non)
Backend -> Base de données : Enregistrement du log de fraude si nécessaire
Backend -> Frontend : Retour du résultat à l’utilisateur/admin
